# SCM Notes - GeoExplore

## Merge Conflict Practice

A merge conflict was intentionally simulated to practice conflict resolution.

## Conflict Scenario
Two branches edited the same line in `app/page.tsx`, specifically the header description text.

## Resolution
Git reported a merge conflict when the second branch was merged into `dev`. The conflict markers were reviewed manually, the preferred final text was kept, and the markers were removed.

## Result
The file was tested after resolution and the conflict was successfully resolved.
